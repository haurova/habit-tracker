# Habit Tracker - C# Learning Project

A simple console-based habit tracker application built while learning C# from scratch.

## Getting Started

### Prerequisites

1. Install [.NET SDK](https://dotnet.microsoft.com/download) (version 8.0 or later)
   - Verify installation: `dotnet --version`

### Running the Project

1. Navigate to the project directory:
   ```bash
   cd habit-tracker
   ```

2. Restore dependencies (optional, usually automatic):
   ```bash
   dotnet restore
   ```

3. Run the application:
   ```bash
   dotnet run
   ```

4. Build the application:
   ```bash
   dotnet build
   ```
